Weather App

Приложение для отображения текущей погоды в выбранном городе с использованием OpenWeatherMap API.

Описание

Weather App позволяет пользователю:

Вводить название города

Получать актуальную погоду в этом городе

Отображать температуру в градусах Цельсия и Фаренгейта

Видеть описание погодных условий и соответствующую иконку

Отображать влажность, давление, скорость ветра и “ощущаемую” температуру

Видеть текущую дату

Технологии

HTML5 – структура страницы

Tailwind CSS – стили и адаптивный дизайн

JavaScript – логика работы с API и обновление интерфейса

OpenWeatherMap API – получение данных о погоде

Google Fonts – шрифты Montserrat

Feather Icons – иконки

Функционал

Ввод названия города и получение данных по нажатию кнопки Get или Enter

Динамическое отображение информации о погоде: температура, описание, иконка, влажность, давление, скорость ветра и ощущаемая температура

Сообщения об ошибках при пустом поле или неверном названии города

Отображение текущей даты

Структура проекта

index.html – основной интерфейс приложения

script.js – работа с API и обновление элементов страницы

Tailwind подключен через CDN для стилей

Feather Icons и Google Fonts подключены для визуальных элементов

Использование

Открыть index.html в браузере

Ввести город в поле ввода

Нажать Get или клавишу Enter

Ознакомиться с информацией о погоде
